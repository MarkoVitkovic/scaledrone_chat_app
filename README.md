# Scaledrone Chat App

## Prerequesites

- NodeJS

## Setup


1. Clone this repository to a location of your choosing

```
mkdir ~/gitlab-projects
cd ~/gitlab-projects
git clone https://github.com/MarkoVitkovic/scaledrone_chat_app.git
cd scaledrone_chat_app
```

2. Run the following commands:

```
cd scaledrone_chat_app
npm install
```

3. Run the frontend

```
npm start
```
